# Day 10 - System Hacking

## System Hacking Kya Hota Hai?

System Hacking ek process hai jisme attacker kisi computer system, mobile device ya server par unauthorized access hasil karne ki koshish karta hai.

Cyber Security ke perspective se, System Hacking ko samajhna zaroori hai taaki security professionals vulnerabilities identify karke systems ko secure bana saken.

### Simple Example

Maan lo kisi ghar me bina permission enter karna illegal hai.

Waise hi kisi computer ya device me bina authorization access karna System Hacking kehlata hai.

---

# System Hacking Ka Objective

Attackers alag-alag reasons ke liye systems ko target kar sakte hain:

* Data Chori Karna
* Financial Fraud
* Sensitive Information Access Karna
* Service Disruption
* Unauthorized Control Hasil Karna

Ethical Hackers ka objective in weaknesses ko identify karke fix karna hota hai.

---

# System Hacking Attack Lifecycle

Generally attackers ek process follow karte hain.

## 1. Reconnaissance

Target ke baare me information collect karna.

Examples:

* Public Information
* Domain Details
* Employee Information

---

## 2. Scanning

Target system ke exposed services aur weaknesses identify karna.

Examples:

* Open Ports
* Running Services
* Software Versions

---

## 3. Vulnerability Identification

System me existing security weaknesses ko identify karna.

Examples:

* Weak Passwords
* Outdated Software
* Misconfigurations

---

## 4. Access Attempt

Security weaknesses ka misuse karke unauthorized access lene ki koshish.

---

## 5. Privilege Escalation

Higher permissions hasil karne ki koshish.

Example:

Normal User

↓

Administrator Access

---

## 6. Maintaining Access

Attackers kabhi-kabhi future access ke liye persistence mechanisms use karne ki koshish karte hain.

---

## 7. Covering Tracks

Logs aur traces hide karne ki attempts ki ja sakti hain.

Security teams isi wajah se monitoring aur logging maintain karti hain.

---
