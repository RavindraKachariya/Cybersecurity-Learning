# Day 1 — Networking for Cyber Security

# 01. What is a Network?

## Socho — Real Life Example

Real life me log ek dusre se communicate karte hain:

* Phone calls ☎️
* WhatsApp 💬
* Instagram 📱
* Email 📧

Ye sab networking ki wajah se possible hota hai.

Jaise roads cities ko connect karti hain, waise networks devices ko connect karte hain.

---

# Network kya hota hai?

## Definition

“A network is a group of two or more devices connected together so they can communicate and share resources.”

Simple samajh:

Jab tumhara mobile Wi-Fi se connect hota hai aur internet chalata hai → tum network use kar rahe ho.

---

# Devices kaun kaun se ho sakte hain?

Network me connected devices:

* Computer
* Laptop
* Mobile
* Printer
* Smart TV
* CCTV Camera
* Server

In sabko networking language me “Nodes” bhi kaha jata hai.

---

# 💡 Simple Formula

## Network = Devices + Connections + Rules (Protocols)

### Devices

Jo communicate karte hain.

### Connections

Jis medium se data travel karta hai.

### Rules / Protocols

Communication ke rules.

---

# Network ka Purpose kya hai?

Networks use hote hain:

* File sharing
* Internet access
* Printer sharing
* Communication
* Cloud access
* Gaming
* Video calls

---

# Real Life Example

Instagram use karte time:

* Tumhara mobile = Device / Node
* Internet = Network
* Instagram server = Server

Tumhara data internet ke through Instagram tak travel karta hai.

---

# Networking Cyber Security me Important kyun hai?

Almost har cyber attack network ke through hota hai.

Hackers:

* Data steal karte hain
* Malware bhejte hain
* Passwords capture karte hain
* Systems hack karte hain

Isliye networking cyber security ki foundation hai.

---

# Cyber Security Importance

## ✓ Spotting Attacks

Agar normal traffic samajh aa gaya:
to suspicious activity detect kar sakte ho.

Example:

* Raat 3 baje huge data transfer
* Unknown login
* Suspicious IP

---

## ✓ Protecting Data

Sensitive data:

* Passwords
* Banking info
* Emails
* Personal files

Network ke through travel karta hai.

Security professionals ensure karte hain:

* Encryption
* Safe transfer
* Secure communication

---

## ✓ Securing the Network

Security experts:

* Firewalls lagate hain
* Access control karte hain
* Suspicious traffic block karte hain

---

# 🔐 Remember This

“You cannot defend what you do not understand.”

Networking cyber security ka backbone hai.

---

# 02. How Devices Connect?

Har network ko ek medium chahiye jisse data travel kare.

2 main types:

1. Wired Network
2. Wireless Network

---

# A) Wired Network

## Definition

Devices cables ke through physically connect hote hain.

Data cable ke andar electrical signals ya light pulses ki form me travel karta hai.

---

# Wired Network Examples

* Office computers
* School computer lab
* Banking systems
* Data centers

---

# Cable Types

| Cable Type        | Use                 |
| ----------------- | ------------------- |
| Ethernet Cable    | Offices & homes     |
| Fiber Optic Cable | High speed internet |
| Coaxial Cable     | Old TV systems      |

---

# Ethernet Cable

Most common networking cable.

Features:

* Cheap
* Reliable
* Fast

Mostly offices me use hota hai.

---

# Fiber Optic Cable

Light ke through data travel karta hai.

Features:

* Extremely fast
* Long distance
* Expensive

Used in:

* ISPs
* Undersea internet cables
* Data centers

---

# Coaxial Cable

Older technology.

Mostly:

* Cable TV
* Internet providers

me use hota tha.

---

# Wired Network Features

## ✓ Speed

Very fast ⚡

Data directly cable ke through travel karta hai.

---

## ✓ Stability

Very stable.

Signal drops almost nahi hote.

---

## ✓ Security

More secure 🔐

Attacker ko cable tak physical access chahiye hota hai.

---

## ✗ Disadvantage

Mobility low hoti hai.

Device cable se connected rehna padta hai.

---

# Wired Network Real Life Example

Office me:

* Sab computers Ethernet cable se switch me connect hote hain
* Switch router se connect hota hai
* Router internet provide karta hai

---

# B) Wireless Network

## Definition

Devices radio waves ke through connect hote hain.

No cable needed.

Most common example:
📶 Wi-Fi

---

# Wireless Network Examples

* Home Wi-Fi
* Mobile internet
* Bluetooth headphones
* Hotspot

---

# Wireless Types

| Type      | Use                  |
| --------- | -------------------- |
| Wi-Fi     | Home/office internet |
| Bluetooth | Short range          |
| 4G/5G     | Mobile internet      |
| Satellite | Remote locations     |

---

# Wi-Fi

Router wireless signal bhejta hai.

Nearby devices:

* Mobile
* Laptop
* Smart TV

connect ho jate hain.

---

# Bluetooth

Short distance communication.

Examples:

* Earbuds
* Smart watches
* Wireless keyboard

---

# Mobile Data (4G/5G)

Phone towers ke through internet access.

Examples:

* Jio
* Airtel
* Vi

---

# Satellite Internet

Remote areas me use hota hai.

Jaha cables possible nahi hote.

---

# Wireless Features

## ✓ Mobility

Move freely 🚶

No cable needed.

---

## ✓ Easy Setup

Installation simple hota hai.

---

## ✓ Flexible

Multiple devices easily connect hote hain.

---

# Wireless Risks

## ✗ Signal Issues

Walls aur distance signal weak kar sakte hain.

---

## ✗ Security Risk

Agar Wi-Fi password protected nahi hai:

* Anyone nearby connect ho sakta hai
* Traffic spy ho sakta hai

---

# Real Attack Example

Attacker:

* Fake Wi-Fi bana deta hai
* User connect karta hai
* Data capture ho jata hai

Isko Evil Twin Attack bhi kehte hain.

---

# Wired vs Wireless Comparison

| Feature    | Wired       | Wireless      |
| ---------- | ----------- | ------------- |
| Speed      | Very Fast   | Moderate      |
| Stability  | Stable      | Signal issues |
| Security   | More secure | More risky    |
| Mobility   | Low         | High          |
| Setup Cost | Higher      | Lower         |

---

# Best Use Cases

## Wired Best For

* Banks
* Offices
* Data centers
* Gaming

---

## Wireless Best For

* Homes
* Cafes
* Schools
* Public areas

---

# 🔐 Cyber Security Notes

## Wired Security

Attacker ko:

* Physical cable access chahiye

Therefore:
More secure.

---

## Wireless Security

Weak password ya open Wi-Fi dangerous ho sakta hai.

Always:

* WPA2/WPA3 use karo
* Strong password rakho
* Public Wi-Fi avoid karo

---

# 03. Types of Networks

Networks size aur area ke basis par divide hote hain.

3 main types:

1. LAN
2. MAN
3. WAN

---

# A) LAN — Local Area Network

## Definition

Small area network.

Devices nearby hote hain.

Examples:

* Home Wi-Fi
* Office floor
* School lab

---

# LAN Features

| Feature   | Details            |
| --------- | ------------------ |
| Full Form | Local Area Network |
| Area      | Room / Building    |
| Speed     | Very Fast          |
| Security  | Easy to manage     |

---

# LAN Examples

* Cyber cafe
* Office computers
* College lab
* Home internet

---

# LAN Advantages

## ✓ High Speed

100 Mbps se 10 Gbps tak.

---

## ✓ Easy Management

Small area hone ki wajah se control easy hota hai.

---

## ✓ Better Security

Limited devices hone ki wajah se monitoring easy hoti hai.

---

# LAN Disadvantages

## ✗ Limited Range

Sirf small area cover karta hai.

---

# 🔐 LAN Security Note

Agar attacker office Wi-Fi join kar le:
to wo:

* Printers
* Shared files
* Systems

access kar sakta hai.

---

# B) MAN — Metropolitan Area Network

## Definition

City level network.

Multiple LANs ko connect karta hai.

---

# MAN Examples

* Bank branches
* University campus
* City CCTV systems

---

# MAN Features

| Feature   | Details                   |
| --------- | ------------------------- |
| Full Form | Metropolitan Area Network |
| Area      | City / Campus             |
| Speed     | Moderate to Fast          |
| Security  | Moderate                  |

---

# MAN Advantages

## ✓ Large Coverage

Pure city ko connect kar sakta hai.

---

## ✓ Shared Resources

Different branches same data access kar sakti hain.

---

# MAN Disadvantages

## ✗ Security Complexity

Multiple locations hone ki wajah se risk increase hota hai.

---

# 🔐 MAN Security Note

Agar branch communication encrypted nahi hai:
to attackers data intercept kar sakte hain.

---

# C) WAN — Wide Area Network

## Definition

Largest type of network.

Countries aur continents ko connect karta hai.

---

# Biggest WAN

🌍 Internet

---

# WAN Examples

* International companies
* Banking networks
* Global cloud services

---

# WAN Features

| Feature   | Details           |
| --------- | ----------------- |
| Full Form | Wide Area Network |
| Area      | Country / World   |
| Speed     | Variable          |
| Security  | Hardest to secure |

---

# WAN Advantages

## ✓ Global Connectivity

Worldwide communication possible.

---

## ✓ Remote Access

Different countries ke offices connect ho sakte hain.

---

# WAN Disadvantages

## ✗ Expensive

Infrastructure costly hota hai.

---

## ✗ Security Challenges

Data multiple unknown systems se pass hota hai.

---

# 🔐 WAN Security Note

Without:

* VPN
* HTTPS
* Encryption

attackers data read kar sakte hain.

---

# LAN vs MAN vs WAN

| Feature  | LAN       | MAN      | WAN       |
| -------- | --------- | -------- | --------- |
| Area     | Building  | City     | World     |
| Speed    | Very Fast | Moderate | Variable  |
| Security | Easy      | Medium   | Difficult |
| Cost     | Low       | Medium   | High      |

---

# Quick Revision

| Term     | Meaning               |
| -------- | --------------------- |
| Network  | Connected devices     |
| Wired    | Cable connection      |
| Wireless | Radio wave connection |
| LAN      | Small area network    |
| MAN      | City network          |
| WAN      | Worldwide network     |

---

# Important Cyber Security Points

## Always Remember

### Public Wi-Fi Risks

* Fake hotspots
* Password theft
* Traffic sniffing

---

### Use Strong Wi-Fi Passwords

Weak passwords easily crack ho sakte hain.

---

### Encryption Important hai

HTTPS aur VPN secure communication provide karte hain.

---

### Networking = Cyber Security Foundation

Networking strong hoga:
to security bhi strong hogi.

---

# Final Thought

Cyber Security me successful banna hai to:
Networking deeply samajhna mandatory hai.

Kyuki:

* Attacks networks se aate hain
* Data networks me travel karta hai
* Protection bhi network level par hoti hai

## 🔐 Golden Rule
