# Part 1

# 📡 Introduction to Wireless Networking

## What is Wireless Networking?

Wireless Networking ek technology hai jo devices ko **without cable** network se connect karti hai.

Communication radio waves ke through hoti hai.

---

## Easy Example

Wired Network:

```text
Laptop
   │
   │ Cable
   │
Router
```

Wireless Network:

```text
Laptop ))))
          ))))
        WiFi
          ))))
Router
```

No cable required.

---

## Definition

Wireless Networking allows devices to communicate using radio signals instead of physical cables.

---

# Why Wireless Networking?

### Mobility

Kahin bhi move kar sakte ho.

### Easy Installation

Cable lagane ki zarurat nahi.

### Flexible

New devices easily connect ho jate hain.

### Cost Effective

Large offices me wiring cost kam ho jati hai.

---

## Real Life Examples

✔ Home WiFi

✔ Mobile Hotspot

✔ Bluetooth Earbuds

✔ Smart Watch

✔ NFC Payments

# Part 8

# 🛡️ Wireless Security Solutions

Wireless networks hackers ke favourite target hote hain.

Isliye security bahut important hai.

---

# 1. Use WPA3 or WPA2

Always use:

✔ WPA3

or

✔ WPA2 AES

Avoid:

❌ WEP

❌ WPA TKIP

---

# 2. Disable WPS

WPS = Wi-Fi Protected Setup

---

Problem:

Attackers brute force attack kar sakte hain.

---

Recommendation:

```text
Disable WPS
```

---

# 3. Strong WiFi Password

Bad Example:

```text
12345678
password
wifi123
```

---

Good Example:

```text
L0ng$Strong@123
```

---

# 4. Update Router Firmware

Updates fix:

✔ Bugs

✔ Security Vulnerabilities

✔ Exploits

---

# 5. Network Segmentation

Separate networks create karo.

Example:

```text
Guest WiFi

Main WiFi
```

Visitors ko main network access na mile.

---

# 6. Monitor Connected Devices

Regularly check:

```text
Who is Connected?
```

---

Tools:

✔ Fing

✔ Router Dashboard

---

# 7. Disable SSID Broadcast

SSID = WiFi Name

Example:

```text
Home_WiFi
```

Hide kar sakte ho.

---

Note:

Professional hackers fir bhi detect kar sakte hain.

---

# 8. Reduce Signal Range

Signal ghar ke bahar kam jana chahiye.

Benefits:

✔ Less Exposure

✔ Better Security

---

# 9. Use VPN

Public WiFi use karte waqt.

Example:

```text
Airport WiFi
```

VPN data encrypt karta hai.

---

# 10. MAC Address Filtering

Sirf approved devices connect kar sakte hain.

Example:

```text
AA:BB:CC:DD
```

Allowed

---

Limitation:

MAC Spoofing possible hai.

---

# 11. Disable Unused Features

Disable:

❌ Bluetooth

❌ NFC

❌ WiFi Direct

if not required.

---

# 12. IDS / IPS

Enterprise security solution.

---

Examples

✔ Snort

✔ Suricata

✔ Cisco Meraki IDS

---

Purpose:

✔ Attack Detection

✔ Traffic Monitoring

✔ Threat Prevention

# Part 9

# 🔐 Wireless Security Protocols

Wireless Security Protocols WiFi ko secure karte hain.

---

# 1. WEP

## Wired Equivalent Privacy

First wireless security protocol.

---

Encryption:

```text
64-bit / 128-bit
```

---

Problems:

❌ Weak Security

❌ Easy To Crack

❌ Outdated

---

Current Status:

```text
Never Use WEP
```

---

# 2. WPA

## Wi-Fi Protected Access

Developed:

```text
2003
```

---

Uses:

```text
TKIP
```

Encryption:

```text
128-bit / 256-bit
```

---

Better than WEP.

But now outdated.

---

# 3. WPA2

Developed:

```text
2004
```

---

Uses:

```text
AES Encryption
```

---

Benefits:

✔ Strong Security

✔ Easy Configuration

✔ Most Common

---

Recommended for old routers.

---

# 4. WPA3

Latest WiFi Security Protocol.

---

Benefits

✔ Strongest Security

✔ Better Password Protection

✔ Better Encryption

✔ WiFi 6 Support

---

Encryption

```text
192-bit
256-bit
384-bit
```

---

Current Recommendation

```text
Use WPA3
```

---

# Part 10

# ⚔️ WEP vs WPA vs WPA2 vs WPA3

| Feature       | WEP       | WPA      | WPA2      | WPA3           |
| ------------- | --------- | -------- | --------- | -------------- |
| Release       | 1999      | 2003     | 2004      | 2018           |
| Security      | Very Weak | Moderate | Strong    | Very Strong    |
| Encryption    | RC4       | TKIP     | AES       | Advanced AES   |
| Easy to Crack | Yes       | Possible | Difficult | Very Difficult |
| Recommended   | No        | No       | Yes       | Best           |

---

# Easy Memory Trick

```text
WEP

↓

WPA

↓

WPA2

↓

WPA3
```

Security continuously improves.
